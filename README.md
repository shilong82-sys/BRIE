## 理论框架（BRIE假说）

```
环境光不足 → 视锥色素亚稳态噪声 → 虚假ON信号
→ ON/OFF失衡（OFF相对过度激活） → 脉络膜变薄 → 眼轴增长
```

**核心干预**：提高"环境光比例"（成像光中环境光成分的占比），恢复ON/OFF 1:1平衡。
