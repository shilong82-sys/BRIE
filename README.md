# BRIE — 高境光理论白皮书

> **Background-Retina Image Environment (BRIE) Hypothesis**
> 近视是成像光与背景环境光长期失耦合导致的视网膜统计稳态偏移

---

## 高境光理论核心框架

```
环境光不足 → 视锥色素亚稳态噪声 → 虚假ON信号
→ ON/OFF失衡（OFF相对过度激活） → 脉络膜变薄 → 眼轴增长
```

**核心干预**：提高"环境光比例"，恢复ON/OFF 1:1平衡。
