## 2.2 ON/OFF通路失衡机制（续）

### 2.2.4 中心-周边拮抗放大机制（续）

**理论推导**（Predicted）：

> ON/OFF通路存在"中心-周边拮抗"结构，每个ON神经元的激活可触发周围18-22个OFF抑制性中间神经元，形成放大效应。

这一结构的机制如下：

| 结构层级 | 机制 | 证据等级 |
|---------|------|---------|
| ON神经元激活 | 谷氨酸释放→下一级神经元激活 | **Established** |
| OFF抑制性中间神经元 | GABA介导的侧向抑制 | **Established** |
| 周边抑制范围 | 每个ON神经元→18-22个OFF神经元 | **Suggested** (推测量级) |
| 净效应 | 总ON/OFF比值决定信号方向 | **Hypothesized** |

**关键假说**：

当总ON/OFF比值 < 1:1时：

1. ON通路活动相对减弱
2. OFF通路相对增强
3. 产生净"近视信号"（myopic signal）
4. 传递给下游脉络膜-巩膜系统

### 2.2.5 ON/OFF失衡与屈光发育的因果关系

**现有证据支持**[Pardue 2008]：

- 形觉剥夺（FFR）可逆地降低ON反应
- ON反应降低与屈光向近视漂移相关
- 提示ON通路活动与屈光发育存在因果关系

**现有证据支持**[Chakraborty 2015]：

- 恢复正常视觉可恢复ON/OFF平衡
- 屈光发育也可恢复正常
- 提示ON/OFF失衡是可逆的

## 2.3 从信号失衡到眼轴增长

### 2.3.1 多巴胺的中介作用

**已有研究表明**[Shu 2023 Nat Neurosci]：

- ON通路激活可促进多巴胺释放（D1受体）
- 多巴胺释放↑ → 抑制眼轴增长
- D1受体敲除 → 近视加速

**机制链条**：

```
ON通路激活 → D1受体激活 → 多巴胺释放↑ → 抑制眼轴增长
ON通路减弱 → D1受体激活↓ → 多巴胺↓ → 近视加速
```

| 环节 | 证据 | 证据等级 |
|------|------|---------|
| ON通路→多巴胺 | Shu 2023 | **Established** |
| 多巴胺→眼轴 | 大量动物实验 | **Established** |
| 完整链条 | 推断 | **Supported** |

### 2.3.2 脉络膜-巩膜级联

**已有研究表明**[Zhou 2021 Prog Retin Eye Res]：

- 脉络膜厚度与眼轴长度呈负相关
- 脉络膜血流↓ → 巩膜缺氧 → I型胶原合成↓
- 巩膜重塑 → 眼轴增长

| 环节 | 证据 | 证据等级 |
|------|------|---------|
| ON/OFF失衡→脉络膜 | Aleman 2018 | **Supported** |
| 脉络膜→眼轴 | 多项队列研究 | **Established** |
| 完整链条 | Zhou 2021 | **Established** |

**关键链接**[Aleman 2018 Sci Rep]：

- 白底黑字（ON刺激）→脉络膜变薄16μm
- 黑底白字（OFF刺激）→脉络膜增厚10μm
- 提示ON/OFF通路活动直接影响脉络膜厚度

### 2.3.3 ipRGCs-昼夜节律的调制作用

**已有研究表明**[Wang 2026]：

- Opn4基因敲除（ipRGCs缺陷）→LAN（室内低照度低光）不再诱导近视
- 提示ipRGCs介导的光信号在低照度近视中发挥关键作用

**理论推导**（Hypothesized）：

ipRGCs可能通过以下途径调制低照度近视：

1. **光强度感知**：ipRGCs对环境照度进行持续监测
2. **昼夜节律同步**：ipRGCs调节褪黑素分泌节律
3. **多巴胺调节**：ipRGCs调节多巴胺能无长突细胞活动
4. **信号放大**：将环境光信息放大至全视网膜

| 机制 | 证据 | 证据等级 |
|------|------|---------|
| ipRGCs→光感知 | Berson 2002 | **Established** |
| ipRGCs→多巴胺 | 内源性连接 | **Supported** |
| ipRGCs��低照度近视 | Wang 2026 | **Suggested** (初步) |
| 完整链条 | 推断 | **Hypothesized** |

### 2.3.4 完整的信号传递链条

综合以上分析，**BRIE理论提出**完整的信号传递链条：

```
室内低照度（<300 lux）
    ↓
视锥暗噪声正向偏移
    ↓
基线偏移型ON激活（假性ON信号）
    ↓
ON/OFF通路失衡（比值<1:1）
    ↓
多巴胺释放减少（D1受体激活↓）
    ↓
脉络膜变薄（脉络膜血流↓）
    ↓
巩膜重塑（I型胶原↓）
    ↓
眼轴增长
```

| 环节 | 证据等级 |
|------|---------|
| 低照度→暗噪声偏移 | **Supported** |
| 暗噪声偏移→ON激活 | **Hypothesized** |
| ON激活→ON/OFF失衡 | **Supported** |
| ON/OFF失衡→多巴胺↓ | **Established** |
| 多巴胺↓→脉络膜变薄 | **Supported** |
| 脉络膜变薄→眼轴增长 | **Established** |

## 2.4 核心数学指标：RSBI

### 2.4.1 RSBI双层定义

#### Level 1：理论RSBI（用于机制讨论）

$$RSBI_{theoretical} = \frac{ON_{effective}}{OFF_{effective}}$$

其中：
- $ON_{effective} = S_{ON} + N_{dark\_bias}$（ON有效信号 = ON信号 + 暗噪声偏移）
- $OFF_{effective} = S_{OFF}$（OFF有效信号）
- $N_{dark\_bias} > 0$（暗噪声偏移为正向）

#### Level 2：测量RSBI（用于实验）

$$RSBI_{measured} = frac{|PhNR_{ON}|}{|PhNR_{OFF}|}$$

其中：
- $PhNR_{ON}$ = 光负反应ON成分振幅（RETeval测量）
- $PhNR_{OFF}$ = 光负反应OFF成分振幅（RETeval测量）

**测量方法**：

- 设备：RETeval手持式ERG（200ms长时程闪光）
- 刺激范式：分离PhNR-ON / PhNR-OFF
- 参数：RSBI = |PhNR-ON| / |PhNR-OFF|

### 2.4.2 RSBI正常范围（假设，待建立）

基于理论推导和初步实验数据：

| RSBI范围 | 临床意义 | 证据等级 |
|---------|---------|---------|
| 0.9 ~ 1.1 | 正视眼（平衡状态） | **Hypothesized** |
| < 0.9 | 近视风险↑（ON通路相对减弱） | **Predicted** |
| > 1.1 | 近视保护（ON通路相对增强） | **Predicted** |

### 2.4.3 RSBI的工程应用

RSBI作为核心状态变量，可用于：

1. **近视风险筛查**：RSBI < 0.9 → 近视高风险
2. **干预效果评估**：BRIE镜片/照明→RSBI变化
3. **个体化参数优化**：基于RSBI的动态照明调节
4. **临床研究终点**：RSBI作为替代终点

## 本章小结

本章详细阐述了BRIE理论的生物机制，包括：

1. **室内低照度的视网膜信号特征**：视锥暗噪声机制、基线偏移型ON激活
2. **ON/OFF通路失衡机制**：动物/人类证据、中心-周边拮抗放大
3. **信号传递链条**：从光环境到眼轴的完整通路
4. **核心指标RSBI**：双层定义、正常范围、工程应用

**核心假说**：

> 室内低照度 → 视锥暗噪声偏移 → 基线偏移型ON激活 → ON/OFF失衡 → 多巴胺↓ → 脉络膜变薄 → 眼轴增长

**证据等级总结**：

- **Level A（Established）**：多巴胺→眼轴、脉络膜→眼轴
- **Level B（Supported）**：低照度→暗噪声偏移、ON/OFF→多巴胺、脉络膜
- **Level C（Hypothesized）**：暗噪声偏移→ON激活、RSBI预测近视
